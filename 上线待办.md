# 新晴小程序上线待办

## 必须由你确认或提供

- [ ] 正式小程序 AppID：替换 `miniprogram-project/project.config.json` 里的 `touristappid`。
- [ ] 小程序主体认证：确认微信公众平台已完成认证。
- [ ] 服务类目：确认类目与新晴功能一致；建议避免选择医疗诊疗类目，除非已有对应资质。
- [ ] 生产域名：提供正式 API 域名，并确认已备案、HTTPS 证书有效。
- [ ] 微信后台合法域名：配置 `request` 域名；如上线后启用上传、下载或 WebSocket，同步配置对应域名。
- [ ] 微信后台隐私保护指引：按实际能力填写手机号、媒体选择、相册写入、反馈处理、账号注销等个人信息处理说明。
- [ ] 用户协议与隐私政策最终文本：确认小程序内文案和微信后台填写内容一致。
- [ ] 客服/反馈联系方式：确认“设置 - 意见反馈”最终承接方式，建议准备邮箱或客服微信。
- [ ] 审核测试账号：如果登录后才能体验完整流程，准备手机号/验证码规则或测试账号说明。
- [ ] 生产数据与备份：确认数据库、对象存储、日志和备份策略已经切到生产环境。

## 提交审核时填写

- [ ] 版本说明：新晴 v2.0.0 第一版，提供情绪记录、小记、慢慢说、心情回看、设置与反馈。
- [ ] 审核路径：进入首页 -> 点击“慢慢说”体验聊天 -> 返回首页 -> 点击“小记”填写内容 -> 点击“我的”查看设置、协议和隐私政策。
- [ ] 登录说明：如果使用测试验证码，提交审核备注中写清楚测试手机号和验证码；正式短信接入后删除测试验证码逻辑。
- [ ] 特殊说明：新晴是日常陪伴与记录工具，不提供医疗诊断、心理治疗或危机干预服务。
