- [ ] 访问 `https://manliaoxiaoji.com/api/health` 返回 `ok: true`、`environment: production`。
- [ ] 微信公众平台已配置合法域名：
  - request 合法域名：`https://manliaoxiaoji.com`
  - uploadFile 合法域名：`https://manliaoxiaoji.com`
  - downloadFile 合法域名：`https://manliaoxiaoji.com`
- [ ] 微信开发者工具打开 `miniprogram-project`。
- [ ] AppID 为 `wx1ae47edde7eb61e8`。
- [ ] `miniprogram-project/config/api.js` 默认环境为 `prod`。

## 一、基础启动

- [ ] 开发者工具编译成功，无红色报错。
- [ ] 首页展示“慢聊小记”入口和底部“此刻 / 我的”导航。
- [ ] 首次进入时展示登录/游客入口。
- [ ] 点击游客模式后可进入首页。

## 二、微信登录

- [ ] 清空缓存后重新进入小程序。
- [ ] 点击登录，微信登录成功。
- [ ] 进入“我的”，显示已登录状态。
- [ ] 关闭并重新打开小程序后仍保持登录。
- [ ] 退出登录后，不展示上一账号的小记、聊天和日历数据。

## 三、慢慢说

- [ ] 登录后进入“慢慢说”。
- [ ] 首次进入自动创建或恢复会话。
- [ ] 空消息点击发送不会提交，并显示提示。
- [ ] 输入一条普通消息并发送。
- [ ] 页面先显示用户消息，再显示后端回复。
- [ ] 连续快速点击发送不会重复提交多条。
- [ ] 重新进入聊天页后能看到历史消息。
- [ ] 进入聊天搜索页，能搜索刚才的消息。
- [ ] 进入聊天日历页，页面可正常返回。

## 四、小记

- [ ] 进入“小记”。
- [ ] 空内容点击“收好”不会提交，并显示提示。
- [ ] 输入一条文字小记并保存成功。
- [ ] 连续快速点击“收好”不会创建重复小记。
- [ ] 进入“我的小记”，能看到刚保存的小记。
- [ ] 点击列表项进入详情，内容一致。
- [ ] 修改详情并保存，返回列表后再次进入能看到更新内容。
- [ ] 删除小记后，列表不再显示该记录。

## 五、图片上传和预览

- [ ] 在“小记”中添加一张图片。
- [ ] 保存小记时显示上传中状态。
- [ ] 保存成功后进入详情，图片能正常显示。
- [ ] 点击图片可预览。
- [ ] 弱网或上传失败时有可理解提示。
- [ ] 拒绝相册/媒体选择权限时，基础文字小记仍可使用。

## 六、心情日历和历史

- [ ] 保存小记后，历史页心情日历对应日期有记录。
- [ ] 删除该日期所有小记后，日历摘要更新。
- [ ] 进入小记搜索页，能搜索到对应内容。
- [ ] 退出登录后，历史和日历不显示旧用户数据。

## 七、设置

- [ ] 进入“我的 - 设置”。
- [ ] 隐私政策页面可打开、可返回。
- [ ] 意见反馈页面可提交一条测试反馈。
- [ ] 反馈提交中按钮 loading，重复点击不会重复提交。
- [ ] 账号注销页可打开，展示风险说明。

## 八、账号注销

- [ ] 使用微信登录账号进入“账号注销”。
- [ ] 未绑定手机号账号显示无需短信验证码。
- [ ] 确认注销后提示成功，并返回首页。
- [ ] 注销后再次进入小记、聊天、历史，不显示旧数据。
- [ ] 注销后旧登录态失效，需要重新登录。

## 九、真机覆盖
