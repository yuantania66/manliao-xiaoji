每晚在固定时间醒来，伴随对白天言行的反复回想和对明天的担忧，即使理智上知道这些想法缺乏依据，身体依然处于紧绷状态；而到了白天，又需要耗费精力维持如常的表现，以避免被他人视为脆弱。这种夜间无法停止的思维反刍与白天的自我掩饰，构成了你当前睡眠困扰的具体样貌。

候选 D：
夜里反复想那些事，身体又紧着，白天还要撑着像平常一样，听起来真的很耗人。

候选 E：
白天维持如常的样子，和夜里身体不由自主地紧绷、反复回想细节，这两者之间似乎隔着一段很难跨越的距离。清醒时你能辨认出那些担忧缺乏依据，但到了那个固定的时间点，身体好像有它自己的记忆和节奏，并不完全听从理智的安抚。这种知道却停不下来的状态，本身就是一种很具体的负重。
